Hi! I'm Spotify Bot and I'm a bot based on music and song search on spotify! All my commands start with the prefix "S/"!

Spotify Bot

Main Commands

Command Description
S/song (Song Name) Search for a song on Spotify.
S/artists (Artists Name) Search for an artist on Spotify.
S/album Search for an album on Spotify.
S/feedback You can give the devs your feedback on the bot! And suggest new things for it! [upvoter only]
S/donate The money donated will go towards paying for a domain so you can make SpotifyBot a website as well as buying a vps from it.
S/help Displays basic information or help for a command!
You can join our support server with S/invite
